# Priage Consultation Website

Marketing and consultation site for Priage — built for hospitals, partners, and patients to learn about the platform and book demos.

## Run locally

```bash
npm install
npm run dev
```

Open **http://localhost:5180**

## Build for production

```bash
npm run build
npm run preview
```

## Pages

| Route | Page |
|-------|------|
| `/` | Home |
| `/for-hospitals` | For Hospitals |
| `/for-patients` | For Patients |
| `/about` | About Us |

## Features

- **Book a demo** modal (header + CTAs) — name, email, message, role dropdown
- **Request a meeting** footer on every page — email capture
- Social icons (email, LinkedIn, Instagram) — placeholders, no links yet
- App Store / Google Play badges — placeholders, no links yet

## Brand colors

Teal `#26C6DA`, blue `#1E88E5`, navy `#0F1729` — from the Priage logo.
